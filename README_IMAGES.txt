AURACARS IMAGE NAMES

Put your images inside folder: images/

Required image names:
1) hero-car.png      -> big hero image
2) dacia.png         -> Dacia car image
3) renault.png       -> Renault car image
4) bmw.png           -> BMW car image
5) clio4.png         -> Clio 4 car image
6) mercedes.png      -> Mercedes car image

If an image is missing, the website will show a placeholder automatically.
